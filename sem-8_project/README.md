## 🚀 Quick Start: Molecular Docking Visualization
To visualize the 2023 Ibuprofen-COX2 docking research automatically:
1. Ensure **UCSF Chimera** is installed.
2. Update the `CHIMERA_PATH` in your `.env` file.
3. Run the following command from the root:
   ```bash
   python visualize_docking.py
